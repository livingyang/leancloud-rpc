# leancloud-rpc
leancloud rpc project.

## How to test
* `cd leancloud`
* `lean up`
* `npm test`

## How to use
1. They are three titles at `leancloud/src/*.ts` file Header:
    * FOR BOTH
    * FOR CLIENT
    * FOR SERVER
2. Copy `FOR BOTH & FOR SERVER` to leancloud project.
3. Copy `FOR BOTH & FOR CLIENT` to client project.
